功能

  • [x] 控件
    • [x] 中国象棋库
      • [x] 中国象棋基本规则
      • [x] 棋局保存和复盘
    • [x] 中国象棋 MFC 扩展动态库
    • [x] 中国象棋 ActiveX 控件
    • [x] 中国象棋 Qt 插件
  • [ ] 中国象棋博弈引擎与通信协议
  • [x] 中国象棋程序
    • [x] 单机版本
      • [x] MFC 版本
      • [x] Qt 版本
    • [ ] 联网版本。欢迎使用本项目库应用到你的中国象棋程序。
